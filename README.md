# Starhold

A pastel space tower defense game — hold the line at the edge of a pastel galaxy.

Built with Vite + TypeScript, no backend required.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Outputs a static site to `dist/`, ready to deploy anywhere (Vercel, etc.).

## Deployment

This repo is the single source of truth for Starhold. Every push to `main` is deployed live via Vercel.
